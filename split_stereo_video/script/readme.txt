Run extract_left_right.sh in bash to extract left and right channels of a given stereo video.

Dependencies - mplayer and matlab should be installed. 
If your matlab binary is not in $PATH then you can set its path in extract_left_right.sh
_________________________________________________________________

Usage
_________________________________________________________________

./extract_left_right.sh ${stereo_video} ${output_directory}
Example - ./extract_left_right.sh chair001.m2ts /tmp/chair001/
